★ Core Modules

Everything in one file.

A single Python script (3,961 lines) containing every module — no imports, no pip install, no setup. Written for Python 3.8+ on any OS.

◆

Mock Exchange Server

Threaded HTTP server on :8080 simulating a financial exchange. Serves real-time order books, tickers, trade execution, and account balances for BTC/USD, ETH/USDT, and more. Rate-limited with exponential backoff.

HTTPServerThreadingREST API
📖

SQLite Ledger Engine

Double-entry bookkeeping with 9 seeded accounts (cash, crypto, revenue, fees). Tracks every trade, verifies debits = credits, and calculates P&L per symbol — all in SQLite with zero setup.

Double-EntrySQLiteP&L
⚙

AST Static Code Auditor

Parses your solution with Python's ast module. Detects: missing try-except on network calls, O(n2) nested loops, eval/exec, division by zero, missing type annotations, infinite while loops, and global state pollution.

ASTStatic AnalysisSecurity
📊

Evaluation Engine

EOD pipeline: reads user code, runs AST audit, executes hidden test suite (subprocess), verifies SQLite ledger integrity, calculates score (0–100). Critical violations = automatic 0. Must score 100 to advance.

SubprocessScoringValidation
💻

Web UI Dashboard

Threaded HTTP server on :8081 serving a full dark-themed dashboard. Live ticker feed, code editor with save/run, evaluation results history, ledger explorer, AST audit viewer — all with auto-polling.

HTML/CSS/JSREST APIDark Theme
📝

Workspace Generator

Generates per-day workspace folders with DAILY_BRIEF.md, boilerplate .py files, and hidden test suites. Days 1–3 have hand-crafted curriculums; days 4–90 use procedural generation.

File GenerationCurriculum

⚡ Tech Stack

Standard library only.

No package manager. No requirements.txt. No node_modules. Built with what ships with Python.

Threaded HTTP servers for exchange & web UI

🗃

ast

Abstract Syntax Tree parsing for code audits

🗃

sqlite3

Embedded database for double-entry ledger

🔌

threading

Background daemon threads for both servers

📜

urllib / json

HTTP client for exchange API calls + JSON parsing

subprocess
