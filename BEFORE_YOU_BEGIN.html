📈 Concept Deep-Dive: Lists & Slicing

You will frequently work with sliding windows of price data. List slicing is essential.

# Sliding window of last 5 prices prices = [100.0, 101.5, 102.3, 101.8, 103.2, 104.0, 103.5] # Last 5 elements window = prices[-5:] # [101.8, 103.2, 104.0, 103.5] # SMA(5) calculation sma_5 = sum(prices[-5:]) / 5 # List comprehension for SMA values def sma(data, window): return [sum(data[i-window:i])/window if i >= window else None for i in range(len(data)+1)][1:]
Day 1 AST: NESTED_LOOP

⚠ Common Mistakes That Will Fail You

❌ Using eval() or exec(): The AST auditor flags these as critical violations (−40 pts). Never use them.
❌ O(n²) nested loops: for i in range(len(p)): for j in range(len(p)): triggers a warning (−10 pts). Use sliding windows instead.
❌ Division by zero: Always guard a / b with if b != 0: or use try/except ZeroDivisionError.

Exception Handling

This is the most critical skill. The AST auditor gives a −40 CRITICAL violation for every network call without a try-except wrapper. Two such violations = automatic 0 for the day.

⚠ The Rule

Every single urllib.request call must be wrapped in try/except. The exchange can go down, return rate limits, or send malformed data. Your pipeline must survive all of it.

import urllib.request import urllib.error def fetch_orderbook(url: str) -> Optional[dict]: try: with urllib.request.urlopen(url, timeout=5) as resp: return json.loads(resp.read().decode()) except urllib.error.HTTPError as e: if e.code == 429: time.sleep(1) # rate-limited, back off return None except urllib.error.URLError as e: print(f"Network error: {e.reason}") return None except json.JSONDecodeError as e: print(f"Bad JSON: {e}") return None except socket.timeout: print("Connection timed out") return None

SQL & SQLite

Every trade, calculation, and audit event is logged to a double-entry SQLite database. You must be comfortable with basic CRUD operations, aggregation, and transaction integrity.

🗃 SQLite in Python

SQLite is embedded in Python's standard library — zero setup required. The database is a single file (training_ledger.db).

import sqlite3 # Connect (creates file if it doesn't exist) conn = sqlite3.connect("training_ledger.db") cursor = conn.cursor() # Create table cursor.execute('''CREATE TABLE IF NOT EXISTS trades ( id TEXT PRIMARY KEY, day INTEGER, symbol TEXT, side TEXT, price REAL, quantity REAL, total REAL, fee REAL, timestamp TEXT )''') # Insert cursor.execute("INSERT INTO trades VALUES (?, ?, ?, ?, ?, ?, ?, ?, ?)", (order_id, day, symbol, side, price, qty, total, fee, ts)) # Query cursor.execute("SELECT COUNT(*) FROM trades WHERE day=?", (day,)) count = cursor.fetchone()[0] # Always commit and close conn.commit() conn.close()

📖 The Ledger Schema

Your solutions will interact with these tables:

-- Account balances (9 seeded accounts) CREATE TABLE accounts ( code TEXT PRIMARY KEY, -- e.g. '1000', '1100' name TEXT NOT NULL, -- e.g. 'Cash - USD' type TEXT NOT NULL, -- 'asset', 'revenue', 'expense' balance REAL DEFAULT 0.0 ); -- Trade log (every executed trade) CREATE TABLE trades ( id TEXT PRIMARY KEY, day INTEGER, symbol TEXT, side TEXT, price REAL, quantity REAL, total REAL, fee REAL, timestamp TEXT ); -- Double-entry ledger entries CREATE TABLE ledger_entries ( id INTEGER PRIMARY KEY AUTOINCREMENT, trade_id TEXT, account_code TEXT, debit REAL, credit REAL, description TEXT ); -- Daily P&L summary CREATE TABLE daily_pnl ( day INTEGER, symbol TEXT, realized_pnl REAL, total_fees REAL, net_pnl REAL, trade_count INTEGER );

⚠ Common Mistake

⚠ Forgetting to commit: conn.commit() is required after INSERT/UPDATE/DELETE. Without it, your changes disappear when the connection closes. The evaluation engine will find zero trades in the ledger and fail you.

💻 Making Requests

import urllib.request import json # GET request url = "http://localhost:8080/v1/ticker?symbol=BTC/USD" with urllib.request.urlopen(url, timeout=5) as resp: data = json.loads(resp.read().decode()) print(data) # {'symbol': 'BTC/USD', 'price': 67306.92, ...} # POST request (execute trade) body = json.dumps({ "symbol": "BTC/USD", "side": "buy", "quantity": 0.5, "type": "market" }).encode() req = urllib.request.Request( "http://localhost:8080/v1/execute", data=body, headers={"Content-Type": "application/json"} ) with urllib.request.urlopen(req, timeout=5) as resp: result = json.loads(resp.read().decode())

💡 Key Formulas You'll Implement

# SMA - Simple Moving Average def sma(prices: List[float], window: int) -> List[Optional[float]]: result: List[Optional[float]] = [] for i in range(len(prices)): if i < window - 1: result.append(None) else: result.append(sum(prices[i-window+1:i+1]) / window) return result # Sharpe Ratio (annualized) def sharpe_ratio(returns: List[float], risk_free_rate: float = 0.0) -> float: excess_returns = [r - risk_free_rate for r in returns] return (statistics.mean(excess_returns) / statistics.stdev(excess_returns)) * math.sqrt(252) # Kelly Criterion (optimal bet size) def kelly_fraction(win_prob: float, win_loss_ratio: float) -> float: return (win_prob * win_loss_ratio - (1 - win_prob)) / win_loss_ratio
